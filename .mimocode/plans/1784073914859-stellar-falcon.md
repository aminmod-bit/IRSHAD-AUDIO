# IRSHAD — Полная пересборка сайта с нуля

## Концепция
Исламская аудио библиотека IRSHAD — филиал YouTube канала @IRSHAD_info (ar-rad channel). Основная функция: прослушивание аудио лекций по исламской тематике. PWA приложение для Telegram, Android, iOS.

---

## Дизайн-система

### Цветовые темы (2 варианта)

**Стеклянный чёрный:**
- Фон: `#0a0a0c`
- Поверхности: `rgba(255,255,255,.06)` → `rgba(255,255,255,.12)` (glass)
- Текст: `#f5f5f7`, приглушённый: `#8a8f98`
- Акцент: `#5E6AD2` (фиолетовый как на плеере)
- Gold: `#d4a574`
- Границы: `rgba(255,255,255,.08)`

**Стеклянный белый:**
- Фон: `#f5f5f7`
- Поверхности: `rgba(0,0,0,.04)` → `rgba(0,0,0,.08)` (glass)
- Текст: `#0c0c0d`, приглушённый: `#6b6b73`
- Акцент: `#5E6AD2`
- Границы: `rgba(0,0,0,.08)`

### Шрифты
- Основной: Inter (400-700)
- Заголовки: Instrument Serif (italic для акцента)

### Glass-эффекты
- `backdrop-filter: blur(20px) saturate(180%)`
- Полупрозрачные фоны с `rgba`
- Тонкие границы `border: 1px solid rgba(..., .08)`
- Тени: `0 8px 32px rgba(0,0,0,.3)`

### Анимации
- Смена разделов: fade + slideUp (200ms ease)
- Glass-эффект при наведении: backdrop-filter pulse
- Плеер: slideUp из底部

## PWA + Telegram Web App

### vite.config.ts
- `VitePWA` плагин с манифестом
- `registerType: 'autoUpdate'`
- Иконки для Android/iOS

### Telegram Web App
- Подключить `telegram-web-app.js`
- `Telegram.WebApp.ready()`
- `Telegram.WebApp.expand()`
- Цвета: `themeParams.bg_color`, `themeParams.text_color`
- Кнопка "Назад": `Telegram.WebApp.BackButton`

### Android/iOS
- `display: standalone` в манифесте
- `apple-mobile-web-app-capable: yes`
- Safe area insets для notch
- Touch action: manipulation (против зума)

## Порядок реализации

1. Удалить старые файлы src/ (App.jsx, index.css, компоненты)
2. Создать package.json с зависимостями
3. Создать vite.config.ts с PWA
4. Создать index.html
5. Создать public/favicon.svg (логотип)
6. Создать public/data/playlists.json
7. Создать src/index.css (glass-дизайн, темы)
8. Создать src/main.jsx
9. Создать src/utils.js
10. Создать src/hooks/useAudio.js
11. Создать src/components/ (все компоненты)
12. Создать src/App.jsx (главный layout)
13. npm install
14. npm run dev — проверить
15. npm run build
16. Запушить на GitHub
17. GitHub Actions задеплоит

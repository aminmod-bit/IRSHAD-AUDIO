# IRSHAD: Обновление UI — баннер, логотип, иконки

## Задача
1. Баннер YouTube как герой-баннер (обложка сверху)
2. Логотип IRSHAD (I с растением) для favicon и sidebar
3. Статичные SVG иконки вместо эмодзи

---

## 1. Баннер YouTube
- Скачать баннер с YouTube канала и положить в `public/banner.jpg`
- В `Home.jsx`: заменить пустой `.hero-bg` на `<img src="/banner.jpg">`
- В `index.css`: `.hero-bg` → `background-image: url('/banner.jpg'); background-size: cover; background-position: center;`
- Добавить полупрозрачный оверлей поверх для читаемости текста

## 2. Логотип IRSHAD
- Создать SVG логотип "I с растением" в `public/favicon.svg`
- Обновить `TopBar.jsx`: заменить inline SVG на новый логотип
- Логотип: элегантная буква I с листком/растением + "IRSHAD" underneath

## 3. Статичные SVG иконки
Создать `src/components/Icons.jsx` с готовыми SVG-иконками:
- Home, Music, Search, Heart, Info (навигация)
- Play, Pause, SkipBack, SkipForward (плеер)
- Repeat, RepeatOne, Volume, VolumeX, List (плеер)
- Sun, Moon (тема)
- Globe, Camera, Send, Phone, Play (ссылки)

Заменить эмодзи в файлах:
- `Sidebar.jsx` — 5 nav иконок
- `App.jsx` — 5 mobile nav иконок
- `Player.jsx` — 8 иконок плеера
- `FullPlayer.jsx` — 6 иконок плеера
- `TopBar.jsx` — 3 иконки (тема, telegram)
- `Search.jsx` — search + favorite иконки
- `PlaylistDetail.jsx` — favorite иконки
- `Favorites.jsx` — favorite иконки
- `About.jsx` — 10 иконок ссылок

## 4. Данные playlists.json
- Заменить emoji в `cover` на текстовые метки (или оставить как есть — эмодзи в данных допустимы, главное не в UI-контролах)
